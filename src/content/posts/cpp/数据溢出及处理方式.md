c++中int可能存在溢出问题，例如：

$$
m = [(i+j)/2]int类型上溢出
$$
当i，j均为int类型时，可能出现int类型上溢出，可考虑使用一下手段在不改变数据类型的前提下，避免溢出：
$$
m = [i+(j-i)/2]
$$


```c++
#include<iostream>
using namespace std;

int main(void){
    int a = 1 << 31, b = 1 << 31;
    a -= 10;
    b -= 1;
    cout << a << " " << b << endl;
    cout << (a+b)/2 << endl;
    cout << a + (b-a)/2 << endl;
    return 0;
}
```
